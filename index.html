Introdução

JavaScript (JS) é uma linguagem de programação de alto nível, interpretada e baseada em eventos. Ele é usado principalmente para criar interatividade em páginas web, sendo uma das três principais tecnologias da web, junto com HTML e CSS.

Principais Caracteristicas

  • Linguagem de script: Executada diretamente no navegador sem necessidade de compilação.
  • Multiparadgma: Suporta programação orientada a objetos, funcional e imperativa.
  • Assíncrona e baseado em eventos: Permitindo operações como requisições HTTP sem bloquear a página
  • Executa no lado do cliente: Embora também possa ser usado no backend com o Node.js

Exemplo Básico

console.log("Olá, Mundo!")

Oque é JavaScript

JavaScript é uma linguagem de programação dinâmica e interpretada, usada principalmente para criar interatividade em páginas web. Ele nos permite modificar elementos do HTML e CSS em tempo real, além de lidar com eventos, animações e requisições para servidors.

Caracteristicas do JavaScript

  • Lado do Cliente (Client-Side): Executado diretamente no navegador, sem necessidade de instalação extra.
  • Lado do Servidor (Server-Side): Também pode ser usado no backend com Node.js.
  • Baseado em Eventos: Reage a ações do usuario, como cliques e digitação.
  • Orientado a Objetos: Usa protótipos em vez de classes tradicionais.

Exemplo Simples

document.getElementById("meuBotao").addEventListener("click", function(){
alert('Você clicou no botão!')
})

JavaScript e Java

JavaScript é uma linguagem de programação interpretada, usada principalmente para desenvolvimento web. Ela é executada no navegador e permite a criação de interfaces dinâmicas, manipulação do DOM e comunicação com servidores via APIs. Além disso, pode ser usada no backend com o Node.js.

Java é uma linguagem de programação orientada a objetos e compilada, usada em aplicações web, desktop, mobile (Android) e servidores. É conhecida por sua portabilidade, já que o código pode rodar em qualquer sistema com a JVM (Java Virtual Machine).

JavaScript

  • Paradigma: Multiparadigma (principalmente funcional e orientado a objetos)
  • Execução: Interpretado, roda no navegador e no servidor (com Node.js)
  • Tipagem: Dinâmica e fraca

Java

  • Paradigma: Orientado a objetos
  • Execução: Compilado para bytecode e executado na JVM
  • Tipagem: Estática e forte
// JavaScript document.getElementById("meuBotao").addEventListener("click", function() { alert("Você clicou no botão!"); }); // Java public class HelloWorld { public static void main(String[] args) { System.out.println("Olá, mundo!"); } }

O que são variáveis em JavaScript

Variáveis são "caixinhas" que guardam valores na memória para serem usados depois no código. Você pode guardar números, textos, listas, objetos e muito mais. No JavaScript, usamos as palavras-chave var, let e const para criar variáveis

Tipos das Variáveis

  • var: forma antiga, tem escopo de função
  • let: forma moderna, tem escopo de bloco
  • const: constante, não pode ser reatribuída
// Criando variáveis com let e const let nome = "Maria" // pode mudar depois const idade = 25 // constante, não muda console.log("Nome:", nome) console.log("Idade:", idade) // Mudando o valor de nome nome = "João" console.log("Novo nome:", nome) // Tentando mudar idade (vai dar erro) // idade = 30 // ❌ Isso causaria um erro pois const não pode ser alterado // Trabalhando com diferentes tipos let ativo = true // booleano let saldo = 1050.75 // número let hobbies = ["ler", "jogar", "cozinhar"] // array console.log("Está ativo?", ativo) console.log("Saldo:", saldo) console.log("Hobbies:", hobbies)

Oque são as Constantes

Constantes são variáveis declaradas com a palavra-chave const que não podem ser reatribuídas depois de criadas.

Use const quando você não pretende mudar o valor da variável depois de declarar. É uma prática comum preferir const sempre que possível, por segurança e clareza no código.

Exemplo

// Declarando uma constante const a = 'nome' const b = 0 const c = {} const d = [] const e = true // true/false

Tipo de Dados

Em JavaScript, os tipos de dados são divididos em duas categorias principais: primitivos e estruturais (ou complexos).

Tipos Primitivos

  • string: texto
  • number: números (inteiros ou com casas decimais)
  • boolean: verdadeiro ou falso
  • undefined: valor não atribuído

Tipos Estruturais

  • object: estrutura de dados com chave-valor
  • array: lista ordenada de valores (também é um tipo de objeto)
  • function: funções também são objetos em JS
